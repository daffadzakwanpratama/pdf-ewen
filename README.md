# PicToPDF Suite - Client-Side PDF Tools

PicToPDF Suite adalah aplikasi web modern, premium, dan sangat interaktif yang memungkinkan pengguna untuk mengolah dokumen PDF secara instan dan 100% aman langsung di browser.

Aplikasi ini berjalan sepenuhnya di **sisi klien (client-side)** menggunakan browser Anda, yang berarti file dan dokumen Anda tidak akan pernah dikirim atau diunggah ke server mana pun di internet. Keamanan dan privasi dokumen Anda terjamin 100%.

🌐 **Demo Live**: [daffadzakwanpratama.github.io/pdf-ewen](https://daffadzakwanpratama.github.io/pdf-ewen/)

## Fitur Utama (3 Tool Utama)

1. **Image to PDF**:
   - Konversi gambar PNG, JPG, JPEG, WebP ke satu dokumen PDF.
   - Pilihan ukuran halaman (A4, Letter, Original), margin, orientasi, dan fitur crop/perspektif pemindai dokumen.
2. **PDF to JPG**:
   - Ekstrak seluruh halaman dari dokumen PDF ke gambar JPG berkualitas tinggi (hingga 300 DPI).
   - Unduh per halaman atau paket ZIP sekaligus.
3. **Compress PDF**:
   - Kecilkan ukuran file PDF langsung di browser tanpa mengurangi keterbacaan dokumen.
   - Pilihan tingkat kompresi (Sedang, Maksimum, Ringan) dan resolusi target.

## Teknologi Yang Digunakan

- **HTML5** & **CSS3 Modern** (Custom CSS Variables, CSS Grid, Flexbox)
- **Vanilla JavaScript** (ES6+)
- **jsPDF** (Pustaka pembuat PDF client-side)
- **pdf.js** (Pustaka pembaca & pengolah PDF oleh Mozilla)
- **JSZip** (Kompresi file ZIP)
- **Lucide Icons** & **Google Fonts** (Plus Jakarta Sans)

## Cara Menjalankan Secara Lokal

1. Unduh atau klon repositori ini:
   ```bash
   git clone https://github.com/daffadzakwanpratama/pdf-ewen.git
   ```
2. Buka file `index.html` dengan mengekliknya dua kali atau gunakan Live Server di VS Code.
